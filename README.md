# Typescript Tetris

[react-tetris][react-tetris-link] 프로젝트에 영감을 받아서, 제작했습니다.
React를 사용하지 않았고, Typescript로 만들었습니다.
테트리스를 만들려는 분들에게 도움이 되길 바랍니다.

[react-tetris-link]: https://github.com/chvin/react-tetris

# 실행방법

## 요구사항
1. Node
2. Chrome Browser

## Module설치
```
npm install
```

## 실행
```
npm run start
```

## Build
```
npm run build
```
